# Тактический компендиум

Веб-приложение для сессий D&D 2024: приёмы Мастера боя, справочник терминов боя оружием, общие команды-шаблоны для ИИ-мастера, бросок костей и интерактивная карта.

## Запуск

Открой `index.html` в браузере, либо разверни через GitHub Pages / Netlify — тогда приложение можно будет установить на телефон как обычное приложение (Add to Home Screen).
